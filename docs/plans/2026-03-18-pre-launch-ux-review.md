# Pre-Launch UX Review Report

**Date:** 2026-03-18
**Scope:** 5 Workspace Types + Shared Components + Entry Pages
**Methodology:** Code-based user journey audit

---

## Executive Summary

经过对 5 种 workspace（thesis, sci, proposal, software_copyright, patent）的完整用户旅程审查，共发现 **47 个 UX 问题**。按优先级分类：

| Priority | Count | Description |
|----------|-------|-------------|
| P0 (阻断上线) | 5 | 功能缺失或严重体验问题 |
| P1 (强烈建议修复) | 12 | 显著影响用户体验 |
| P2 (建议优化) | 18 | 中等影响，可后续迭代 |
| P3 (锦上添花) | 12 | 轻微影响 |

## P0 Issues (阻断上线)

### P0-1: 导出功能为占位符
**位置:** `compile-export/page.tsx`
**问题:** 导出格式按钮（PDF、Word、LaTeX、Markdown）只是占位符，没有实际功能
**影响:** 用户点击后无任何反应，造成困惑
**建议:**
- 要么实现导出功能
- 要么暂时移除按钮或显示"即将推出"提示

### P0-2: 图表生成章节关联硬编码
**位置:** `figure-generation/page.tsx`
**问题:** 章节关联下拉框硬编码为"第一章"到"第四章"，不反映实际大纲
**影响:** 如果大纲有 5+ 章，用户无法正确关联
**建议:** 从 thesis_writing store 或 artifacts 动态获取章节列表

### P0-3: 文献管理"添加文献"按钮无功能
**位置:** `literature/page.tsx`
**问题:** "添加文献"按钮没有 onClick 处理，点击无反应
**影响:** 用户无法手动添加文献
**建议:** 实现添加文献 modal 或至少显示 toast 提示功能开发中

### P0-4: 文献管理"筛选"按钮无功能
**位置:** `literature/page.tsx`
**问题:** "筛选"按钮没有 onClick 处理
**影响:** 用户无法筛选文献列表
**建议:** 实现筛选功能或移除按钮

### P0-5: 智能盘点积分系统未解释
**位置:** `literature/page.tsx`
**问题:** "智能盘点（20积分）"按钮提到积分，但积分系统从未在界面其他地方解释
**影响:** 用户不知道积分从哪来、怎么用
**建议:**
- 移除积分显示（如果暂不实现积分系统）
- 或添加 tooltip 解释积分系统

## P1 Issues (强烈建议修复)

### P1-1: 主页面语言混合
**位置:** `workspaces/[id]/page.tsx`
**问题:** `workspaceTypeLabels` 是英文（"Thesis / Dissertation"），但界面其他部分是中文
**影响:** 视觉不一致，降低专业感
**建议:** 统一为中文标签

### P1-2: 多数功能页无历史记录展示
**位置:** 所有功能页面
**问题:** 除了 literature 页面，其他功能页不显示历史执行记录或产出
**影响:** 用户不知道之前执行过什么、产出在哪里
**建议:** 在主区域添加"历史记录"或"最近产出"区块

### P1-3: 任务执行无进度指示
**位置:** 所有使用 `useFeatureTaskRunner` 的页面
**问题:** 长时间任务只显示"任务执行中..."，没有进度条或预计时间
**影响:** 用户不知道任务要等多久，可能中途离开
**建议:** 添加进度阶段指示（如"检索中... → 分析中... → 生成中..."）

### P1-4: Step 2 禁用状态不明显
**位置:** `thesis-writing/page.tsx`
**问题:** "Step 2: 全文写作" 在无大纲时禁用，但只有 opacity-50，视觉不够明确
**影响:** 用户可能不理解为什么不能点击
**建议:** 添加 tooltip 说明"请先生成大纲"

### P1-5: 编译导出页无编译历史
**位置:** `compile-export/page.tsx`
**问题:** 编译失败时用户看不到错误日志，成功时看不到历史编译
**影响:** 调试困难，用户不知道失败原因
**建议:** 添加"编译历史"区块，显示最近几次编译状态和日志

### P1-6: 返回按钮行为不一致
**位置:** 所有功能页
**问题:** 返回按钮总是返回 `/workspaces/${workspaceId}`，而不是浏览器历史
**影响:** 用户期望"返回上一页"，实际跳转到 workspace 主页
**建议:** 使用 `router.back()` 或添加面包屑导航

### P1-7: 开题调研主区域空状态
**位置:** `opening-research/page.tsx`
**问题:** 主区域是占位符，不显示生成的历史报告
**影响:** 用户需要返回 workspace 主页查看产出
**建议:** 在主区域添加历史报告列表

### P1-8: 图表生成主区域无预览
**位置:** `figure-generation/page.tsx`
**问题:** 主区域是占位符，不显示已生成的图表
**影响:** 用户看不到产出，需要去其他地方找
**建议:** 添加已生成图表的预览区

### P1-9: 论文写作无草稿保存提示
**位置:** `thesis-writing/page.tsx`
**问题:** 用户可能不知道进度是自动保存的
**影响:** 用户担心丢失进度
**建议:** 添加"进度已自动保存"提示

### P1-10: 上下文 Artifact ID 输入不友好
**位置:** `writing/page.tsx` (sci)
**问题:** 用户需要手动输入 Artifact ID，但不知道 ID 是什么
**影响:** 功能几乎无法使用
**建议:** 改为下拉选择器，显示可用的 artifacts 列表

### P1-11: thesis workspace 缺少底部 panels
**位置:** `workspaces/[id]/page.tsx`
**问题:** thesis workspace 不显示 Knowledge/Chat/Literature panels，但其他 workspace 类型显示
**影响:** thesis 用户无法快速访问这些功能
**建议:** 统一所有 workspace 类型的布局，或解释差异原因

### P1-12: Workspace 列表页统计数为 0
**位置:** `workspaces/page.tsx`
**问题:** `paperCount={0}` 和 `artifactCount={0}` 硬编码，不反映实际数据
**影响:** 卡片上的统计信息无意义
**建议:** 从 API 获取真实数据

## P2 Issues (建议优化)

### P2-1: 章节内容无富文本渲染
**位置:** `thesis-writing/page.tsx`
**问题:** 章节内容用 `whitespace-pre-wrap` 显示纯文本
**影响:** 无法展示格式化内容（加粗、列表等）
**建议:** 添加 Markdown 渲染

### P2-2: 无"导出章节"功能
**位置:** `thesis-writing/page.tsx`
**问题:** 用户无法单独导出某一章节
**影响:** 需要整篇导出才能获取内容
**建议:** 添加"导出本章"按钮

### P2-3: 论文分析页参数输入冗余
**位置:** `paper-analysis/page.tsx`
**问题:** Paper ID、标题、摘要都需要手动输入
**影响:** 用户体验繁琐
**建议:** 支持从文献列表一键跳转并自动填充

### P2-4: 无 workspace 使用引导
**位置:** 所有 workspace
**问题:** 新用户不知道使用流程（先做哪步后做哪步）
**影响:** 可能跳过必要步骤
**建议:** 添加步骤引导或流程图

### P2-5: 模块卡片状态可能不准确
**位置:** `ModuleCard.tsx`
**问题:** 状态依赖 dashboard 数据，可能不同步
**影响:** 显示"未开始"但实际已执行过
**建议:** 添加手动刷新或轮询机制

### P2-6: 任务状态反馈位置不统一
**位置:** 各功能页
**问题:** 有的在侧边栏底部，有的在主区域
**影响:** 视觉不一致
**建议:** 统一放置位置

### P2-7: 空状态文案不统一
**位置:** 各功能页
**问题:** 有的有"暂无XX"，有的直接显示占位符
**影响:** 体验不一致
**建议:** 统一空状态设计

### P2-8: 无快捷键支持
**位置:** 所有功能页
**问题:** 不支持 Cmd+Enter 提交等快捷键
**影响:** 效率用户不满意
**建议:** 添加常用快捷键

### P2-9: 无任务取消功能
**位置:** 所有执行任务的页面
**问题:** 任务开始后无法取消
**影响:** 误操作无法撤销
**建议:** 添加"取消任务"按钮

### P2-10: 无表单验证提示
**位置:** 多个表单页面
**问题:** 必填字段只有红点标记，提交时不提示具体错误
**影响:** 用户不知道漏填了什么
**建议:** 添加表单验证和错误提示

### P2-11: 时间范围选项不一致
**位置:** `background-research/page.tsx` vs `prior-art-search/page.tsx`
**问题:** 两个页面的时间范围选项不同（近3/5/10年 vs 近1/3/5/10年/不限）
**影响:** 体验不一致
**建议:** 统一时间范围选项

### P2-12: 技术说明书加载状态无取消
**位置:** `technical-description/page.tsx`
**问题:** 加载默认配置时整个表单不可见
**影响:** 如果加载失败，用户看到空白
**建议:** 只在必要字段显示加载态

### P2-13: 产出物卡片无预览功能
**位置:** `RecentArtifacts.tsx`
**问题:** 点击产出物卡片无反应
**影响:** 用户需要去其他地方查看详情
**建议:** 添加点击预览或跳转

### P2-14: 无 workspace 搜索结果为空提示
**位置:** `workspaces/page.tsx`
**问题:** 搜索无结果时只显示空状态，没有"无匹配结果"提示
**影响:** 用户不知道是搜索无结果还是真的没有 workspace
**建议:** 区分两种空状态

### P2-15: 删除确认用 confirm()
**位置:** `workspaces/page.tsx`
**问题:** 删除确认使用浏览器原生 confirm，风格与整体 UI 不一致
**影响:** 视觉突兀
**建议:** 使用自定义确认 modal

### P2-16: 无 workspace 排序功能
**位置:** `workspaces/page.tsx`
**问题:** 列表无排序选项
**影响:** workspace 多时难以管理
**建议:** 添加按名称/日期排序

### P2-17: 筛选按钮无下拉
**位置:** `workspaces/page.tsx`
**问题:** 筛选按钮没有下拉菜单
**影响:** 按钮无实际功能
**建议:** 实现筛选或移除按钮

### P2-18: 无批量操作
**位置:** `workspaces/page.tsx`
**问题:** 无法批量删除或移动 workspace
**影响:** 管理效率低
**建议:** 添加多选和批量操作

## P3 Issues (锦上添花)

### P3-1: 无暗色模式切换
**位置:** 全局
**问题:** CSS 变量支持暗色模式，但无切换入口
**影响:** 用户无法选择主题

### P3-2: 无国际化切换入口
**位置:** 全局
**问题:** 代码有 i18n 支持，但无 UI 切换入口
**影响:** 非中文用户无法使用

### P3-3: 任务执行无声音反馈
**位置:** 全局
**问题:** 任务完成或失败无提示音
**影响:** 用户可能错过通知

### P3-4: 无 PWA 支持
**位置:** 全局
**问题:** 无离线使用能力
**影响:** 无法作为独立应用使用

### P3-5: 无键盘导航支持
**位置:** 全局
**问题:** 不支持 Tab 键完整导航
**影响:** 无障碍访问受限

### P3-6: 无撤销/重做
**位置:** 表单输入
**问题:** 表单输入不支持撤销
**影响:** 误删内容无法恢复

### P3-7: 无自动保存指示
**位置:** 表单输入
**问题:** 表单修改后无自动保存提示
**影响:** 用户担心丢失输入

### P3-8: 无分享功能
**位置:** workspace
**问题:** 无法分享 workspace 给他人
**影响:** 协作受限

### P3-9: 无导出报告 PDF 功能
**位置:** 各产出页面
**问题:** 产出无法直接导出 PDF
**影响:** 需要复制粘贴

### P3-10: 无版本历史
**位置:** 产出物
**问题:** 无法查看产出的历史版本
**影响:** 无法回退

### P3-11: 无收藏/置顶功能
**位置:** workspace 列表
**问题:** 无法置顶常用 workspace
**影响:** 需要频繁搜索

### P3-12: 无标签分类
**位置:** workspace
**问题:** 无法给 workspace 打标签
**影响:** 分类管理受限
