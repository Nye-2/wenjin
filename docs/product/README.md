# Product Docs

## 文档索引

- `workspace-feature-catalog.md`: 五类 workspace 的功能目录（以 registry 为准）
- `frontend-feature-plugin-contract.md`: 前后端 feature 契约与交互约束
- `release-gate-checklist.md`: 发布门禁与上线前检查项
- `continuous-optimization-plan.md`: 长期持续优化计划，覆盖全链路、workspace 完整度、技术债务、架构和规范治理
- `workspace-chat-centered-redesign.md`: 以 workspace 为任务容器、以 chat 为驾驶舱的产品改版方案
- `workspace-chat-centered-implementation-plan.md`: 基于当前代码仓库的详细实施任务单、阶段 review 与上线计划
